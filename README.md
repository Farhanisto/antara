# antara
Challenge

#### How to run backend
1. Clone the project
2. ```cd antara && pipenv shell```
3. ```cd app &&  python manage.py makemigrations```
4. python manage.py migrate
5.  python manage.py runserver
6.  visit http://localhost:8000/graphql


#### How to run client

1. ```cd client && yarn ```
2. yarn start
